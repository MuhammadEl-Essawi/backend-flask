🔒 Password Policy

  • Minimum 8 characters
  • Uppercase + Lowercase required
  • At least one digit
  • Werkzeug hash (bcrypt-like)

🔑 OTP Security

  • 6-digit cryptographic generation
  • Hashed storage (never plaintext)
  • 5-minute expiry window
  • Max 5 attempts then lockout

Integration

.NET ↔ Flask Webhooks

Secure bi-directional communication with the .NET admin dashboard using HMAC-SHA256 signed payloads.

# Webhook signature verification def _verify_signature(payload, sig): expected = hmac.new( secret.encode("utf-8"), payload, hashlib.sha256 ).hexdigest() return hmac.compare_digest( expected, sig )
